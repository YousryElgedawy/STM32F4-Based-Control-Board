# STM32F4-Based-Control-Board
STM32F411CE-based board for controlling 4 DC motors using 4× PWM outputs and 2 quadrature encoders via timer-based interrupt inputs. Features 6× analog inputs, I²C, SPI, UART, Bluetooth module interface, and USB 2.0 connectivity. Powered via USB, DC barrel jack, or pin headers with automatic priority switching. Supports SWD programming
